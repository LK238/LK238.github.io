#Introduction: What Stories Is Netflix Telling Us?

Netflix has revolutionized how we consume entertainment, but have you ever wondered how the streaming giant describes its content? The words chosen to summarize movies and TV shows aren’t random—they’re carefully crafted to attract specific audiences while accurately representing the content.

In this analysis, I dive deep into Netflix’s title descriptions to uncover:

What words appear most frequently?

How does sentiment differ across content?

Do TV ratings (TV-MA, TV-14, PG, etc.) use different language?

What happens when negation words like “not” or “never” flip the meaning?

Let’s explore what 8,000+ Netflix descriptions reveal about how we talk about entertainment.

#Bigrams sentiments

Now lets look at the context in which these combination of words appear so that we can see if they actually contribute to a negative or positive sentiment.

Code 
netflix_bigrams_afinn <- netflix_ngram |>
  separate(bigram, c("word1", "word2"), sep = " ") |>
  filter(!word1 %in% stop_words$word,
         !word2 %in% stop_words$word) |>
  count(word1, word2, sort = TRUE) |>
  
  # Add AFINN scores
  left_join(get_sentiments("afinn"), by = c("word1" = "word")) |>
  rename(afinn_word1 = value) |>
  left_join(get_sentiments("afinn"), by = c("word2" = "word")) |>
  rename(afinn_word2 = value) |>
  
  # Create combined sentiment columns
  mutate(
    # Replace NA with 0
    afinn_word1 = replace_na(afinn_word1, 0),
    afinn_word2 = replace_na(afinn_word2, 0),
    
    # Combined scores
    combined_sum = afinn_word1 + afinn_word2,
    
    
    # Sentiment category based on combined score
    combined_sentiment_afinn = case_when(
      combined_sum > 2 ~ "very positive",
      combined_sum > 0 ~ "positive",
      combined_sum == 0 ~ "neutral",
      combined_sum < 0 ~ "negative",
      combined_sum < -2 ~ "very negative"
    )
  )

I filtered out duration-based ratings (like “90 min”) and focused on content ratings: TV-MA, TV-14, TV-PG, R and PG-13.

# First, let's create a bigram dataset with ratings and sentiment
bigrams_with_ratings <- Netflix_Data |>
  filter(!str_detect(rating, "^\\d+ min$")) |>
  filter(rating %in% c("R", "TV-14","PG-13","TV-MA","TV-PG"))|>
  mutate(line = row_number()) |>
  unnest_tokens(bigram, description, token = "ngrams", n = 2) |>
  separate(bigram, c("word1", "word2"), sep = " ", remove = FALSE) |>
  filter(!is.na(word1) & !is.na(word2)) |>
  filter(!word1 %in% stop_words$word & !word2 %in% stop_words$word) |>
  
  # Add sentiment for both words using AFINN
  left_join(get_sentiments("afinn"), by = c("word1" = "word")) |>
  rename(afinn_word1 = value) |>
  left_join(get_sentiments("afinn"), by = c("word2" = "word")) |>

    afinn_word1 = replace_na(afinn_word1, 0),

    bigram_sentiment = case_when(
      combined_score > 0 ~ "positive",

#What have we Learned About Netflix’s Language?

Based on this comprehensive analysis of over 8,000 Netflix titles, we can conclude that Netflix’s description language reveals a sophisticated content strategy that balances emotional appeal with audience targeting. While the raw counts show more negative words (11,465) than positive words (7,919) overall, positive terms like “love,” “best,” and “work” appear more frequently.

The most revealing finding here is how language shifts dramatically by content rating: mature-rated content (R, TV-MA, TV-14) relies heavily on negative sentiment words associated with crime, violence, and psychological tension, while youth-oriented ratings (PG-13, TV-PG) feature more positive and neutral language centered on friendship and adventure. In addition the analysis of negation patterns further demonstrates that words following “never” tend toward negative sentiment while those after “without” lean positive, showing how Netflix uses context to create nuanced descriptions that manage viewer expectations.

Ultimately, these patterns reveal that Netflix’s descriptions are carefully crafted linguistic tools that not only summarize content but strategically signal genre conventions, emotional journeys, and appropriate audiences through deliberate word choices and contextual framing.

#Bigrams sentiments 

Now lets look at the context in which these combination of words appear so that we can see if they actually contribute to a negative or positive sentiment.

```{r}
netflix_bigrams_afinn <- netflix_ngram |>
  separate(bigram, c("word1", "word2"), sep = " ") |>
  filter(!word1 %in% stop_words$word,
         !word2 %in% stop_words$word) |>
  count(word1, word2, sort = TRUE) |>
  
  # Add AFINN scores
  left_join(get_sentiments("afinn"), by = c("word1" = "word")) |>
  rename(afinn_word1 = value) |>
  left_join(get_sentiments("afinn"), by = c("word2" = "word")) |>
  rename(afinn_word2 = value) |>
  
  # Create combined sentiment columns
  mutate(
    # Replace NA with 0
    afinn_word1 = replace_na(afinn_word1, 0),
    afinn_word2 = replace_na(afinn_word2, 0),
    
    # Combined scores
    combined_sum = afinn_word1 + afinn_word2,
    
    
    # Sentiment category based on combined score
    combined_sentiment_afinn = case_when(
      combined_sum > 2 ~ "very positive",
      combined_sum > 0 ~ "positive",
      combined_sum == 0 ~ "neutral",
      combined_sum < 0 ~ "negative",
      combined_sum < -2 ~ "very negative"
    )
  )

# View results
netflix_bigrams_afinn |>
  select(word1, word2, n, afinn_word1, afinn_word2, 
         combined_sum, combined_sentiment_afinn) |>
  filter(afinn_word1 != 0 | afinn_word2 != 0) |>
  head(20)
```

#The Negation Effect: When "Not" Changes Everything

One of the most fascinating aspects of language is how negation words flip meaning. "Good" is positive, but "not good" is negative. Conversely, "bad" is negative, but "not bad" is positive.

Let's see how Netflix uses negation:

```{r}
netflix_afinn <- get_sentiments("afinn")

bigrams_separated <- netflix_ngram |>
  separate(bigram, c("word1", "word2"), sep = " ") |>
  count(word1, word2, sort = TRUE) |>
  filter(!is.na(word1) & !is.na(word2))

bigrams_separated |> filter(word1 == "not")

not_words <- bigrams_separated |>
  filter(word1 == "not") |>
  inner_join(netflix_afinn, by = c(word2 = "word")) |>
  arrange(desc(n))

not_words
```

```{r}

#| fig.alt: "This barplot shows the distribution of the most common words combined with negation words in the Netflix dataset based on their sentimemt (positive or negative). In the x-axis we have the sentiment score of these words  ranging from -8 to 3  and the y-axis we have the words that follow a specific negation word. "
```

#Rating Matters: How Language Differs by TV Rating

Now for the most revealing part of my analysis: How does language change across different TV ratings?

I filtered out duration-based ratings (like "90 min") and focused on content ratings: TV-MA, TV-14, TV-PG, R and PG-13.

#Bigrams: Word Pairs by Rating

```{r}
#| fig.cap: "When analysing how the paired words are distributed across the selected TV rating types, we can see that for R and TV-PG the TV-Shows or Movies have most often paired words with negative or neutral sentiment, whereas for PG-13 and TV-14 we see most often neutral themes."


# First, let's create a bigram dataset with ratings and sentiment
bigrams_with_ratings <- Netflix_Data |>
  filter(!str_detect(rating, "^\\d+ min$")) |>
  filter(rating %in% c("R", "TV-14","PG-13","TV-MA","TV-PG"))|>
  mutate(line = row_number()) |>
  unnest_tokens(bigram, description, token = "ngrams", n = 2) |>
  separate(bigram, c("word1", "word2"), sep = " ", remove = FALSE) |>
  filter(!is.na(word1) & !is.na(word2)) |>
  filter(!word1 %in% stop_words$word & !word2 %in% stop_words$word) |>
  
  # Add sentiment for both words using AFINN
  left_join(get_sentiments("afinn"), by = c("word1" = "word")) |>
  rename(afinn_word1 = value) |>
  left_join(get_sentiments("afinn"), by = c("word2" = "word")) |>
  rename(afinn_word2 = value) |>
  
  # Calculate combined sentiment
  mutate(
    afinn_word1 = replace_na(afinn_word1, 0),
    afinn_word2 = replace_na(afinn_word2, 0),
    combined_score = afinn_word1 + afinn_word2,
    bigram_sentiment = case_when(
      combined_score > 0 ~ "positive",
      combined_score < 0 ~ "negative",
      TRUE ~ "neutral"
    )
  )

# Now count bigrams by rating and sentiment
bigrams_by_rating <- bigrams_with_ratings |>
  count(rating, bigram, bigram_sentiment, sort = TRUE) |>
  group_by(rating) |>
  slice_max(n, n = 5) |>  # Top 5 bigrams per rating
  ungroup()

# Plot
bigrams_by_rating |>
  ggplot(aes(x = fct_reorder(bigram, n), y = n, fill = bigram_sentiment)) +
    geom_col() +
    coord_flip() +
    facet_wrap(~ rating, scales = "free_y") +
    scale_fill_manual(values = c("positive" = "darkgreen", 
                                  "negative" = "darkred", 
                                  "neutral" = "gray50")) +
    labs(
      title = "Most Common Bigrams by TV Rating",
      subtitle = "Colored by combined AFINN sentiment score",
      x = "Bigrams", 
      y = "Frequency",
      fill = "Sentiment"
    ) +
    theme_minimal() +
    theme(
      strip.text = element_text(size = 8, face = "bold"),
      legend.position = "bottom",
      axis.text.y = element_text(size = 7)
    )
  
```

#What have we Learned About Netflix's Language? 

Based on this comprehensive analysis of over 8,000 Netflix titles, we can conclude that Netflix's description language reveals a sophisticated content strategy that balances emotional appeal with audience targeting. While the raw counts show more negative words (11,465) than positive words (7,919) overall, positive terms like "love," "best," and "work" appear more frequently.

The most revealing finding here is how language shifts dramatically by content rating: mature-rated content (R, TV-MA, TV-14) relies heavily on negative sentiment words associated with crime, violence, and psychological tension, while youth-oriented ratings (PG-13, TV-PG) feature more positive and neutral language centered on friendship and adventure. In addition the analysis of negation patterns further demonstrates that words following "never" tend toward negative sentiment while those after "without" lean positive, showing how Netflix uses context to create nuanced descriptions that manage viewer expectations.

Ultimately, these patterns reveal that Netflix's descriptions are carefully crafted linguistic tools that not only summarize content but strategically signal genre conventions, emotional journeys, and appropriate audiences through deliberate word choices and contextual framing.
